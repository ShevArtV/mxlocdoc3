# Диагностика

## Папка документации не найдена

Проверьте `mxlocdoc.docs_path`. Предпочтительный переносимый формат:

```text
[[+corePath]]components/project-docs/
```

Итоговая папка после раскрытия пути должна существовать и быть доступной PHP для чтения.

## Картинки не отображаются

Проверьте, что путь к картинке указан относительно текущего Markdown-файла, а расширение есть в `mxlocdoc.allowed_asset_extensions`.

## Поиск не обновляется

Очистите кэш MODX manager. mxLocDoc очищает свой кэш через plugin `mxLocDocCacheClear`.

## Язык не появился в переключателе

Языковая папка должна быть прямым потомком `mxlocdoc.docs_path`, например `en/` или `ru/`, и внутри должна быть Markdown-документация.
