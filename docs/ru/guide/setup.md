# Установка

Установите mxLocDoc как MODX 2 package и откройте компонент из меню manager.

Пакет создаёт namespace, пункт меню, системные настройки и manager processors. Сами документы остаются обычными Markdown-файлами на диске.

## Минимальная настройка

1. Создайте локальную папку документации.
2. Добавьте `README.md`.
3. Укажите путь к папке в `mxlocdoc.docs_path`.
4. Очистите кэш MODX manager.
5. Откройте mxLocDoc в manager.

## Путь по умолчанию

Значение по умолчанию:

```text
[[+corePath]]components/mxlocdoc/docs/
```

Плейсхолдер `[[+corePath]]` раскрывается в путь к MODX core во время выполнения. Простой относительный путь тоже считается относительно папки core.
