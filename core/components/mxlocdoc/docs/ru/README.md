# mxLocDoc

mxLocDoc — MODX Revolution 3 extra для просмотра локальной Markdown-документации внутри manager.

Пакет рассчитан на проектную документацию, которая должна лежать на том же сервере, что и сайт, и быть доступной пользователям админки без публикации файлов наружу.

## Что умеет

- Читает Markdown-файлы из локальной папки, заданной системной настройкой.
- Строит левую навигацию из `_sidebar.json` или `mxlocdoc.json`.
- Строит fallback-навигацию по файлам, если manifest отсутствует.
- Безопасно рендерит Markdown в manager.
- Отдаёт относительные картинки и разрешённые ассеты через защищённый connector.
- Ищет по Markdown-файлам.
- Поддерживает языковые папки, например `en/` и `ru/`.

## Что не входит

- Редактирование Markdown из manager.
- Публичный фронтовый вывод документации.
- Таблицы БД для документов или поиска.
- Node, Vite, Vue или отдельный frontend build.

## Рекомендуемая структура

```text
core/components/mydocs/
├── en/
│   ├── _sidebar.json
│   ├── README.md
│   └── guide/
└── ru/
    ├── _sidebar.json
    ├── README.md
    └── guide/
```

Настройка `mxlocdoc.docs_path` указывает на родительскую папку:

```text
[[+corePath]]components/mydocs/
```

Если внутри есть языковые папки, mxLocDoc покажет переключатель языка в manager.
