# Руководство

Этот раздел описывает рабочий цикл mxLocDoc: установку, настройки, навигацию, Markdown-файлы, локальные ассеты, поиск, очистку кэша и многоязычную документацию.

Используйте эти страницы, когда нужно подключить папку документации к manager, подготовить manifest навигации, добавить картинки или вложения, включить поиск либо разделить документацию по языкам.
