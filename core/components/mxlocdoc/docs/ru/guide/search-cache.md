# Поиск и кэш

Поиск сканирует Markdown-файлы по заголовку, пути и plain text содержимому.

Подготовленный поисковый индекс кешируется в:

```text
core/cache/mxlocdoc/
```

В ключе кэша учитывается выбранный языковой корень, поэтому у каждого языка свой индекс.

## Очистка кэша

Плагин `mxLocDocCacheClear` слушает событие MODX `OnBeforeCacheUpdate`. Когда пользователь очищает кэш в manager, mxLocDoc удаляет и свой поисковый кэш.

Установите `mxlocdoc.cache_ttl` в `0`, чтобы отключить кеширование поискового индекса.
